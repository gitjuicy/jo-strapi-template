# Juicyorange Strapi Template
This template contains some default fields that should apply to most Juicyorange Strapi projects. 

To start a new project using this template run:
```
npx create-strapi-app my-new-project --template gitjuicy/jo-strapi-template --quickstart
```
You can skip the login and decline A/B testing.
Make sure to create and populate all the .env.example values to your .env file.
